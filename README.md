# Open Citadel

Open Citadel is an open-source, privacy-first knowledge companion app. Collect, organise, and act on insights from books, blogs, and podcasts — turning what you read and hear into decisions, ideas, and progress.

## Features

- **Timeline** — Track your learning journey over time
- **Library** — Organize all your sources in one place
- **Reader** — Read and annotate content with ease
- **Samwell** — Your AI-powered learning assistant

## Tech Stack

Built with Astro, featuring a custom design system with support for light/dark themes.

## Development

```sh
pnpm install
pnpm dev
```

## Commands

| Command          | Action                                           |
| :--------------- | :----------------------------------------------- |
| `pnpm install`   | Installs dependencies                            |
| `pnpm dev`       | Starts local dev server at `localhost:4321`      |
| `pnpm build`     | Build your production site to `./dist/`          |
| `pnpm preview`   | Preview your build locally, before deploying     |
| `pnpm astro ...` | Run CLI commands like `astro add`, `astro check` |
